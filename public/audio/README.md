# AntCarts Audio Assets

These sound effects are converted and trimmed from OpenGameArt `SoundPack01.zip`.

- Source page: https://opengameart.org/content/level-up-power-up-coin-get-13-sounds
- Source archive: https://opengameart.org/sites/default/files/SoundPack01.zip
- License shown on source page: CC0 1.0 Public Domain

Mappings:

- `coin.ogg`: `Coin01.aif`, trimmed to 0.72s
- `mystery-box.ogg`: `FX01.aif`, trimmed to 0.95s
- `powerup.ogg`: `Upper01.aif`, trimmed to 1.15s

The OGG files are shipped from Vite's `public/` directory and played locally
through the browser Web Audio API.
